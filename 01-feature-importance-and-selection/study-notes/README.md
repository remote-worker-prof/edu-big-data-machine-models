# Study Notes (обязательная часть самостоятельной работы)

Эта папка используется для углубленных заметок по методам и подходам,
которые применяются в лабораторной работе.

Когда использовать:
- если объяснение метода не помещается в ноутбук;
- если нужно отдельно зафиксировать сравнение нескольких подходов;
- если вы хотите дать более полное обоснование выбора метода.

Минимальные требования к каждой заметке:
1. Кратко указать, **что изучено**.
2. Пояснить, **где это применимо** в текущей ЛР.
3. Дать **сравнение с альтернативой**.
4. Отразить **ограничения/риски** метода.
5. Указать **источники** (1-2+, URL или литература).

Как ссылаться:
- В ноутбуках и отчёте ставьте явную ссылку на файл в `study-notes/`.
- Формат ссылки: `study-notes/<имя_файла>.md`.

Рекомендуемая схема именования:
- `filter-variance-vs-mi.md`
- `wrapper-rfe-vs-sfs.md`
- `threshold-tuning-practice.md`
